  # Atomic State
  Atomic State is a state management library for React.
</div>

### Overview

Atomic State works very similarly to `useState`, while providing features that make state management easier.

### Features:

- Incrementaly adoptable
- TypeScript-ready
- Server-Side Rendering
- React Native
- Effects
- Reducers
- Selectors (filters)
- Persistence

and [more](/docs/api)!
